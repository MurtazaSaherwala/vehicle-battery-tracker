# Vehicle Battery Tracker

Full Stack MERN application with JWT authentication and battery tracking visualization.

## Tech Stack
- Node.js
- Express
- MongoDB
- React
- Chart.js
- JWT

## Features
- User Authentication
- Vehicle Management
- Battery History Tracking
- Data Visualization
- Protected Routes

## Setup

Backend:
cd backend
npm install
npm run dev

Frontend:
cd frontend
npm install
npm run dev
